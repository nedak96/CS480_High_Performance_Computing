avx is a simple program that uses Intel's AVX intrinsics to show that it runs faster than it would doing it sequentially.  openmp is a simple program that utilizes openmp to show how fast it parallelizes picking random points in a "sphere" of increasing dimensionality and determining the distance from the center.

To compile files type: make
To run avx program: type: ./avx
To run openmp program type: ./openmp
